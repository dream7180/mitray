# mitray

A tray to manage mihomo core on windows.

## Features

- Start/Stop mihomo core
- Enable/Disable system proxy
- Enable/Disable TUN mode
- Check mihomo status
- Open mihomo webui
- Open mihomo directory
- Auto start mihomo on startup
- Auto detect mihomo status

## Usage

1. Download the latest release from [Releases](https://github.com/tom2almighty/mihomo-tray/releases).
2. Extract the files to a directory.
3. Edit `config.ini` and set the path to your mihomo executable and config file.
4. Run `mitray.exe`.

> [!IMPORTANT]
> If you want to use TUN mode, you need to run `mitray.exe` as administrator.
